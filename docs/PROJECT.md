# Football Info — сайт чемпионата мира по футболу 2026

## Что это
Сайт-витрина ЧМ-2026 (США / Канада / Мексика, проходит сейчас — июнь/июль 2026).
Главный фокус — живой матч, расписание и турнирные таблицы.

## Ключевые решения (зафиксированы с заказчиком)
- **Стек:** Next.js (фуллстек) — фронтенд + API-роуты + БД в одном проекте. Деплой на Vercel.
- **Источник данных:** football-data.org, бесплатный тариф. Код турнира — `WC`.
- **Язык интерфейса:** русский.
- **Адаптивность:** обязательна — десктоп и мобильные. Mobile-first.

## Экраны и функциональность

### 1. Главный экран — «Сейчас в игре»
- Показывает один из **актуальных (live) матчей**. Если live несколько — берём любой
  (по умолчанию первый по времени старта).
- **Фон:** флаги двух играющих стран (приём: диагональный градиент-сплит или
  два полупрозрачных флага слева/справа).
- **Карточка матча снизу:** счёт, минута, статус, названия команд, флаги, группа/стадия.
- **Лайв-обновление:** опрос данных раз в ~60 сек, счёт/минута меняются без перезагрузки.
- Если live-матчей нет → fallback: ближайший предстоящий матч или последний завершённый
  (состояние «сейчас матчей нет»).

### 2. Меню / навигация
- **Предстоящие** — все незавершённые матчи (запланированные и идущие), сгруппированы по дням.
- **Завершённые** — все сыгранные матчи, от свежих к старым, сгруппированы по дням.
- **Таблица** — турнирные таблицы:
  - **Групповой этап** — таблицы по группам (A–L для ЧМ-2026, 12 групп).
  - **Плей-офф** — сетка на вылет (1/16 → финал).

## Нефункциональные требования
- Корректная статистика, аккуратная типографика, читаемость.
- Бережём лимит API (10 запросов/мин на free) — кэшируем в своей БД, не дёргаем
  внешний API на каждый клиент-запрос.
- Тёмная/светлая тема — опционально, не в MVP.

## Открытые вопросы (вернуться позже)
- Нужны ли детальные стат-метрики матча (владение, удары)? На free-тарифе их НЕТ
  (см. API-RESEARCH.md). На MVP показываем счёт, минуту, статус, забивших — если доступны.
- Нужна ли страница отдельного матча (клик по карточке) — отложено на v2.
